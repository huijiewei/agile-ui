Agile 组件库的 CSS-IN-JS 方案使用的是 [Twind](https://twind.style/)，样式使用 [Tailwind CSS](https://tailwindcss.com/)

## 定制主题

通过 twind.config.ts 文件配置，可以灵活定制自己的主题

```ts
import { twindConfig } from '@agile-ui/twind';
import { css } from '@twind/core';

export default twindConfig({
  preflight: css`
    body {
      @apply overflow-y-scroll bg-white text-black antialiased;
    }
  `,
  theme: {}, // 通过修改 theme 定制自己的主题
});
```
