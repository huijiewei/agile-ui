---
title: 选择器
description: 当用户需要从一组同类数据中选择一个或多个时，可以使用下拉选择器，点击后选择对应项。
---

## 用法

```tsx preview editable
<div className={'flex flex-row flex-wrap gap-3'}>
  <Select defaultValue={16} clearable placeholder={'选择器'}>
    {Array(20)
      .fill(null)
      .map((_, idx) => idx + 1)
      .map((eve) => (
        <SelectOption key={eve} value={eve}>
          选项 {eve}
        </SelectOption>
      ))}
  </Select>
  <Select className={'w-72'} closeOnSelect={false} multiple clearable placeholder={'多项选择器'}>
    {Array(20)
      .fill(null)
      .map((_, idx) => idx + 1)
      .map((eve) => (
        <SelectOption key={eve} value={eve}>
          选项 {eve}
        </SelectOption>
      ))}
  </Select>
</div>
```

```tsx preview editable
function Demo() {
  const browsers = [
    {
      value: 'chrome',
      name: 'Google Chrome',
      icon: <GoogleChrome />,
      description: '谷歌浏览器',
    },
    { value: 'firefox', name: 'Firefox', icon: <Firefox />, description: '火狐浏览器' },
    {
      value: 'safari',
      name: 'Safari',
      icon: <Safari />,
      description: '苹果浏览器',
    },
    {
      value: 'edge',
      name: 'Microsoft Edge',
      icon: <MicrosoftEdge />,
      description: '一款快速而安全的浏览器，可帮助您保护数据，节省时间和金钱。',
    },
    { value: 'opera', name: 'Opera', icon: <Opera />, description: '比默认浏览器更快、更安全、更智能。' },
    {
      value: 'brave',
      name: 'Brave',
      icon: <Brave />,
      description: '一款快速、私密且安全的网络浏览器。',
    },
    { value: 'vivaldi', name: 'Vivaldi', icon: <Vivaldi />, description: '' },
    { value: 'ie', name: 'Internet Explorer', icon: <InternetExplorer />, description: '' },
  ];

  return (
    <div className={'flex flex-col gap-2'}>
      <Select
        className={'w-56'}
        filter={(keyword, option) => {
          return (
            option.props.extra.value.toLowerCase().indexOf(keyword.toLowerCase()) != -1 ||
            option.props.extra.name.toLowerCase().indexOf(keyword.toLowerCase()) != -1
          );
        }}
        searchable
        placeholder={'浏览器'}
      >
        {browsers.map((browser) => (
          <SelectOption extra={browser} key={browser.value} value={browser.value}>
            <div className={'flex flex-row items-center gap-1.5'}>
              {browser.icon}
              {browser.name}
            </div>
          </SelectOption>
        ))}
      </Select>
      <Select className={'w-full'} searchable multiple placeholder={'浏览器'}>
        {browsers.map((browser) => (
          <SelectOption extra={browser} key={browser.value} value={browser.value}>
            <div className={'flex flex-row items-center gap-1.5'}>
              {browser.icon}
              {browser.name}
            </div>
          </SelectOption>
        ))}
      </Select>
    </div>
  );
}
```
