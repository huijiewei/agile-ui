---
title: 选择器
description: 当用户需要从一组同类数据中选择一个或多个时，可以使用下拉选择器，点击后选择对应项。
---

## 用法

```tsx preview editable
<div className={'flex flex-row flex-wrap gap-3'}>
  <Select defaultValue={16} placeholder={'选择器'}>
    {Array(20)
      .fill(null)
      .map((_, idx) => idx + 1)
      .map((eve) => (
        <SelectOption key={eve} value={eve}>
          选项 {eve}
        </SelectOption>
      ))}
  </Select>
  <Select closeOnSelect={false} multiple placeholder={'多项选择器'}>
    {Array(20)
      .fill(null)
      .map((_, idx) => idx + 1)
      .map((eve) => (
        <SelectOption key={eve} value={eve}>
          选项 {eve}
        </SelectOption>
      ))}
  </Select>
</div>
```

## 大数据

大数据虚拟列表选择器的触发条件有两个: 子元素全部为 SelectOption 组件；子元素数量超过 30 个。

```tsx preview editable
<div className={'flex flex-row gap-3'}>
  <Select className={'w-60'} placeholder={'选择器'}>
    {Array(1000)
      .fill(null)
      .map((_, idx) => idx + 1)
      .map((eve) => (
        <SelectOption key={eve} value={eve}>
          选项 {eve}
        </SelectOption>
      ))}
  </Select>
</div>
```
