---
title: 选择器
description: 当用户需要从一组同类数据中选择一个或多个时，可以使用下拉选择器，点击后选择对应项。
---

## 用法

```tsx preview editable
<div className={'flex flex-row flex-wrap gap-3'}>
  <Select defaultValue={16} clearable placeholder={'选择器'}>
    {Array(20)
      .fill(null)
      .map((_, idx) => idx + 1)
      .map((eve) => (
        <SelectOption key={eve} value={eve}>
          选项 {eve}
        </SelectOption>
      ))}
  </Select>
  <Select className={'w-72'} closeOnSelect={false} multiple clearable placeholder={'多项选择器'}>
    {Array(20)
      .fill(null)
      .map((_, idx) => idx + 1)
      .map((eve) => (
        <SelectOption key={eve} value={eve}>
          选项 {eve}
        </SelectOption>
      ))}
  </Select>
</div>
```
