---
title: 选择器
description: 当用户需要从一组同类数据中选择一个或多个时，可以使用下拉选择器，点击后选择对应项。
---

## 用法

```tsx preview editable
<Select placeholder={'选择器'}>
  <SelectOption className={'whitespace-nowrap'} value={'1'}>
    选项一一一一一一一一一一一一一一一一一一
  </SelectOption>
  <SelectOption value={'2'}>选项二</SelectOption>
  <SelectOption value={'3'}>选项三</SelectOption>
  <SelectOption value={'4'}>选项四</SelectOption>
  <SelectOption value={'5'}>选项五</SelectOption>
</Select>
```
